# galvis-post1-u1
Laboratorio 1 — Configuración del entorno de desarrollo web

## Descripción del proyecto
Este proyecto es un sitio web de ejemplo que se utiliza para configurar el entorno de desarrollo web. El sitio incluye una estructura básica de HTML, CSS y JavaScript, y se puede utilizar como base para futuros proyectos web.

## Estructura del proyecto
El proyecto tiene la siguiente estructura de archivos:
```galvis-post1-u1/
├── index.html
├── css/
│   └── styles.css
└── js/
    └── script.js
```
## Configuración del entorno de desarrollo
Para configurar el entorno de desarrollo, sigue estos pasos:
1. Clona el repositorio del proyecto en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web para ver el sitio en acción.
3. Edita los archivos `styles.css` y `script.js` para personalizar el sitio web según tus necesidades.
4. Utiliza herramientas de desarrollo web como Chrome DevTools para depurar y mejorar tu sitio web. 
5. Asegúrate de guardar tus cambios y actualizar el navegador para ver los resultados.

## Comandos de ejecución
No se requieren comandos de ejecución específicos para este proyecto, ya que es un sitio web estático. Simplemente abre el archivo `index.html` en tu navegador para ver el sitio en acción.
